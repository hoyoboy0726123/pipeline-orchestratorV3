# Skill 沙盒容器預裝套件
# 分成幾個 tier：build 時 Dockerfile 會拆成多個 RUN 讓每層獨立裝，避免一次性 OOM
#
# 沒進這清單但某個 workflow 需要的套件，可以事後臨時裝進容器：
#   wsl sudo docker exec pipeline-sandbox pip install <pkg>
# 要永久加，把它寫進這檔然後 rebuild image。
